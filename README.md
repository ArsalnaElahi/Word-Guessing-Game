# Word-Guessing-Game
An interactive word guessing game built in Python with difficulty selection, smart hint system, and limited attempts for enhanced gameplay.

## How It Works

- Choose a difficulty: easy, medium, or hard  
- A secret word is randomly selected  
- You have 7 attempts to guess the word  
- After each wrong guess, a hint shows correct letters in correct positions  

## Requirements

- Python 3.x

## How to Run

1. Download or clone the repository
2. Open a terminal in the project folder
3. Run:

```bash
python game.py
